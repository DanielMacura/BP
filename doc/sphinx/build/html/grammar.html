Grammar

class grammar.Production(LHS, RHS, nullable=False, increment_number=True)

Productions are rules that consist of a left-hand-side(LHS) and right-hand-side(RHS). The LHS is a single NonTerminal and the RHS is a list of Terminals and NonTerminals to replace the LHS when the production is applied.

We can specify a production in the following maner.

>>> Production(a, [b,c], nullable=True)
>>> Production(a, [d,e], nullable=True)
Parameters:

  • LHS (NonTerminal) – Left-hand-side of the production.

  • RHS (Union[List[NonTerminal | Terminal], Epsilon]) – Right-hand-side of the production.

  • nullable (bool) – Specifies if the production is nullable.

class grammar.Grammar

The Grammar is defined by a list of productions. It allows multiple operations, such as adding rules, checking if a NonTerminal is nullable and returning all terminals.

Parameters:

productions – List of productions defining a grammar.
