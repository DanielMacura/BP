class precedence_table.PrecedenceTable

The PrecedenceTable provides methods to interact with the precedence table. A precedence table is usually hand written and later loaded from a file for use in the parser.

Parameters:

  • symbols – List of all symbols the precedence table accepts.

  • symbol_names – Mappings of symbol class names, to be stored in the precedence file.

  • precedence_table – The table has the same index and columns, a square shape.
